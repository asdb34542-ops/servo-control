# servo-control
Servo Motors Control Panel

تم إنشاء واجهة ويب للتحكم بأربع محركات سيرفو باستخدام HTML, CSS, JavaScript, PHP مع تخزين الزوايا في قاعدة بيانات MySQL.
الخطوات
	1.	تشغيل XAMPP وتفعيل Apache وMySQL.
	2.	إنشاء قاعدة بيانات باسم servo_db وجدول لتخزين الزوايا.
	3.	إنشاء مجلد باسم servo4_control داخل htdocs.
	4.	إضافة ملفات المشروع وربطها بقاعدة البيانات
  ثم نقوم بتشغيل من المتصفح 
  http://localhost/servo4_control/
